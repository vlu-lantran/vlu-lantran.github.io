Câu 1: Thuật toán Naive Bayes

Đoạn mã mẫu dưới đây sử dụng thư viện sklearn để huấn luyện mô hình Multinomial Naive Bayes và dự đoán cho mẫu e6.

#Naive Bayes classifier
from sklearn.naive_bayes import MultinomialNB
import numpy as np

# Training data
e1 = [2, 1, 2, 2, 2, 2]
e2 = [2, 2, 2, 1, 2, 2]
e3 = [1, 0, 2, 1, 2, 0]
e4 = [2, 0, 1, 1, 2, 1]
e5 = [1, 1, 1, 2, 0, 1]
e6 = [2, 2, 0, 1, 1, 2]
train_data = np.array([e1, e2, e3, e4, e5, e6])
ket_qua = np.array(['yes', 'yes', 'no', 'no', 'no', 'yes'])

# Test data
test_1 = [2, 1, 2, 1, 2, 0]
test_2 = [1, 0, 1, 2, 2, 1]
test_data = np.array([test_1, test_2])

ml = MultinomialNB(alpha=1)
ml.fit(train_data, ket_qua)
print('Probability of test_data :\n', ml.predict_proba(test_data))
print('Predicting class of test_data :', ml.predict(test_data))

Yêu cầu lập trình: Hãy viết mã Python (không sử dụng hàm fit hay predict_proba của sklearn cho phần tính toán chính) để thực hiện từng bước tính toán chi tiết cho quá trình TRAINING và TEST. Mã của bạn cần tính toán và in ra màn hình các giá trị sau:

  • Bước 1: Xác suất tiên nghiệm (Prior probability) của lớp 'yes' và lớp 'no'.
  • Bước 2: Xác suất hậu nghiệm (Posterior probability) cuối cùng của mẫu e6. Đối chiếu kết quả với đầu ra [0.63078213 0.36921787].

Câu 2: Cập nhật Q-Learning

Cho một mê cung có kích thước 5x5, vị trí bắt đầu (0,0), vị trí kết thúc (4,4) và danh sách các chướng ngại vật như trong đoạn mã mẫu.

#Đọc cột trước - hàng sau
maze_size = 8           #kích thước mê cung
obstacles = [(3,0),(1,1),(3,1),(1,2),(3,3), (4,5), (5,5), (6,2), (6,3), (2,6), (7,1)]   #vị trí vật cản
start = (0,0)           #bắt đầu
goal = (7,7)            #kết thúc

#kiểm tra vị trí nhất định của (x,y) có hợp lệ di chuyển hay không
def is_valid(x,y):
    return (0 <= x < maze_size) and (0 <= y < maze_size) and (x,y) not in obstacles

Yêu cầu lập trình: Hãy viết mã Python để thực hiện quá trình khởi tạo và cập nhật một bước tính toán của Q-Learning dựa trên phương trình Bellman.
